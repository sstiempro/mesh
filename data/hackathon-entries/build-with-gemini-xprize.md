### Title
Mesh: Autonomous Crypto-Intelligence Engine for Education and Productivity

### One-liner
Mesh is an open-source, keyless crypto-intelligence and security engine that utilizes machine learning and autonomous capabilities to provide real-time monitoring and insights, aligning with the themes of Machine Learning/AI, Education, and Productivity.

### Problem
The cryptocurrency and DeFi spaces are plagued by a lack of transparency, security risks, and information overload, making it challenging for users to stay informed and make data-driven decisions. Existing solutions often rely on proprietary data sources, closed systems, and manual intervention, limiting their effectiveness and accessibility.

### What we built
Mesh is a shipped and running system, with a verifiable public commit history on GitHub. The project includes:
* ~98 live keyless monitors running on a 24/7 cron, providing early warnings for stablecoin depeg, protocol TVL-drain exploits, and other critical events
* An autonomous capability-generation engine (monitor-forge.mjs) that discovers new opportunity categories, writes new monitors using a large language model (LLM), and ships them live with zero human intervention
* A smart-contract audit pipeline using Slither, Aderyn, and Foundry for public-goods security
* A multi-chain wallet supporting EVM, Solana, Bitcoin, Cosmos, Sui, Aptos, and Near addresses derived from a single seed
* An open-core public signal feed (RSS/JSON) with a free tier for the commons and a pro tier for sustainability
* 100% keyless and free public data sources, ensuring reproducibility and accessibility

### Why it fits this hackathon's themes
Mesh aligns with the Machine Learning/AI theme through its use of LLMs for autonomous monitor generation and its application of machine learning concepts to crypto-intelligence. The project also fits the Education theme by providing a transparent, open-source, and accessible platform for users to learn about cryptocurrency and DeFi. Additionally, Mesh contributes to the Productivity theme by offering a suite of tools and insights that help users make informed decisions and stay up-to-date with market developments.

### Tech stack
Mesh is built using a variety of technologies, including Node.js, JavaScript, and large language models. The project leverages public data sources such as DefiLlama, Hyperliquid, CoinGecko, Snapshot, and Deribit, as well as public RPCs.
